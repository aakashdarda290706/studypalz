Operators in Python
What are Operators?

Operators are special symbols that perform operations on variables and values. Python provides a variety of operators for different types of operations.

1. Arithmetic Operators
Operator Description Example
+ Addition 5 + 3 = 8
- Subtraction 5 - 3 = 2
* Multiplication 5 * 3 = 15
/ Division 5 / 3 = 1.666...
// Floor Division 5 // 3 = 1
% Modulus 5 % 3 = 2
** Exponentiation 5 ** 3 = 125
Arithmetic Operators Example
x = 10
y = 3

print(f"Addition: {x + y}")        # 13
print(f"Subtraction: {x - y}")     # 7
print(f"Multiplication: {x * y}")  # 30
print(f"Division: {x / y}")        # 3.333...
print(f"Floor Division: {x // y}") # 3
print(f"Modulus: {x % y}")        # 1
print(f"Exponent: {x ** y}")      # 1000
2. Relational Operators
Operator Description Example
== Equal to 5 == 5 is True
!= Not equal to 5 != 3 is True
> Greater than 5 > 3 is True
< Less than 5 < 3 is False
>= Greater than or equal to 5 >= 5 is True
<= Less than or equal to 5 <= 3 is False
Relational Operators Example
x = 5
y = 3

print(f"Equal to: {x == y}")                 # False
print(f"Not equal to: {x != y}")            # True
print(f"Greater than: {x > y}")             # True
print(f"Less than: {x < y}")                # False
print(f"Greater than or equal to: {x >= y}") # True
print(f"Less than or equal to: {x <= y}")    # False
3. Logical Operators
Operator Description Example
and True if both operands are true True and True is True
or True if at least one operand is true True or False is True
not Inverts the boolean value not True is False
Logical Operators Example
x = True

Python Math Module

Python provides a rich set of mathematical functions through the math module. Here are some commonly used functions:

Basic Mathematical Functions
Function Description Example

12. Maximum of Three Numbers

Write a program to find the maximum of three numbers.

Solution
def find_maximum(a, b, c):
    return max(a, max(b, c))

# Get input
num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))
num3 = float(input("Enter third number: "))

# Calculate and display result
maximum = find_maximum(num1, num2, num3)

13. Leap Year Check

Write a program to check if a year is leap year.

Solution
def is_leap_year(year):
    return year % 4 == 0 and (year % 100 != 0 or year % 400 == 0)

# Get input
year = int(input("Enter a year: "))

# Check and display result
if is_leap_year(year):
    print(f"{year} is a leap year")
else:
    print(f"{year} is not a leap year")
14. Valid Date Check

Write a program to check if a date is valid.

Solution
def is_valid_date(day, month, year):
    if month < 1 or month > 12:
        return False
        
    # Days in each month
    days_in_month = [0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]
    
    # Adjust February for leap year
    if month == 2 and (year % 4 == 0 and (year % 100 != 0 or year % 400 == 0)):
        days_in_month[2] = 29
        
    return 1 <= day <= days_in_month[month]

# Get input
day = int(input("Enter day: "))
month = int(input("Enter month: "))
year = int(input("Enter year: "))

# Check and display result
if is_valid_date(day, month, year):
    print(f"{day}/{month}/{year} is a valid date")
else:
    print(f"{day}/{month}/{year} is not a valid date")
15. Quadratic Equation Roots

Write a program to find the roots of a quadratic equation.

Solution
import math

def find_roots(a, b, c):
    # Calculate discriminant
    discriminant = b**2 - 4*a*c
    
    if discriminant > 0:
        root1 = (-b + math.sqrt(discriminant)) / (2*a)
        root2 = (-b - math.sqrt(discriminant)) / (2*a)
        return f"Two distinct roots: {root1:.2f} and {root2:.2f}"
    elif discriminant == 0:
        root = -b / (2*a)
        return f"One repeated root: {root:.2f}"
    else:
        real = -b / (2*a)
        imag = math.sqrt(abs(discriminant)) / (2*a)
        return f"Complex roots: {real:.2f} + {imag:.2f}i and {real:.2f} - {imag:.2f}i"

# Get coefficients
a = float(input("Enter coefficient a: "))
b = float(input("Enter coefficient b: "))
c = float(input("Enter coefficient c: "))

# Calculate and display roots
print(find_roots(a, b, c))
16. Delete Characters at Index Divisible by 3

Write a program to delete all characters from a string whose indices are divisible by 3.

Solution
def remove_chars_at_three(string):
    return ''.join(char for i, char in enumerate(string) if i % 3 != 0)

# Get input
text = input("Enter a string: ")

# Process and display result
result = remove_chars_at_three(text)
print(f"Original string: {text}")
print(f"Modified string: {result}")
17. Widest Equal Elements Fragment

Find the length of the widest fragment where all elements are equal in a sequence of numbers ending with 0.

Solution
def find_widest_equal_fragment():
    numbers = []
    max_length = 1
    current_length = 1
    
    # Get input sequence
    while True:
        num = int(input("Enter a number (0 to end): "))
        numbers.append(num)
        if num == 0:
            break
    
    # Find widest fragment
    for i in range(1, len(numbers)):
        if numbers[i] == numbers[i-1]:
            current_length += 1
            max_length = max(max_length, current_length)
        else:
            current_length = 1
    
    return max_length

# Calculate and display result
widest_length = find_widest_equal_fragment()
print(f"Length of widest equal fragment: {widest_length}")
18. Bowling Pin Simulator

Simulate bowling pins being knocked down and show remaining pins.

Solution
def simulate_bowling():
    # Get initial setup
    N = int(input("Enter number of pins: "))
    K = int(input("Enter number of balls to be rolled: "))
    
    # Initialize pins (all standing)
    pins = ['I'] * N
    
    # Process each ball
    for _ in range(K):
        start = int(input("Enter start position: "))
        stop = int(input("Enter stop position: "))
        
        # Mark pins as knocked down
        for i in range(start-1, stop):
            pins[i] = '.'
    
    # Display final state
    return ''.join(pins)

# Run simulation and display result
result = simulate_bowling()
print(f"Final pin configuration: {result}")
19. Time Difference Calculator

Calculate the number of seconds between two timestamps.

Solution
def calculate_time_difference():
    # Get first timestamp
    print("Enter first timestamp:")
    h1 = int(input("Hours: "))
    m1 = int(input("Minutes: "))
    s1 = int(input("Seconds: "))
    
    # Get second timestamp
    print("Enter second timestamp:")
    h2 = int(input("Hours: "))
    m2 = int(input("Minutes: "))
    s2 = int(input("Seconds: "))
    
    # Convert both timestamps to seconds
    time1 = h1 * 3600 + m1 * 60 + s1
    time2 = h2 * 3600 + m2 * 60 + s2
    
    # Calculate difference
    difference = time2 - time1
    return difference

# Calculate and display result
seconds_between = calculate_time_difference()
print(f"Seconds between timestamps: {seconds_between}")
